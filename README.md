Ms. G's Arithmancy Class

A wizard-themed, gamified multiplication web app built for 3rd grade level math practice.

The Goal: Answer as many multiplication questions as possible in 3 minutes.

The Rules: Questions are strictly limited to numbers 1 through 12.

The Time Turner: There is a 10% chance a question will be a special "Time Turner" card (glowing gold). Answering correctly adds +20 seconds to the timer.

Scoring:

Outstanding: > 50 correct answers.

Exceeds Expectations: > 30 correct answers.

Acceptable: < 30 correct answers.
